# Portfolio Website

This is a personal portfolio website showcasing my professional background, including:

- Introduction
- Work Experience
- Education
- Achievements
- Volunteering Experience

## Getting Started

Simply open `index.html` in your web browser to view the portfolio.

## Customization

To personalize this portfolio:

1. Replace placeholder text in `index.html` with your actual information.
2. Update the contact details in the footer.
3. Modify the CSS styles as needed for your preferred design.

## Technologies Used

- HTML5
- CSS3
